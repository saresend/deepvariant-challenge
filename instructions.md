## Step 1

- Install Gsutil:

```
wget https://storage.googleapis.com/pub/gsutil.tar.gz
tar xfz gsutil.tar.gz -C <dir_path>
```

## Step 2

- Clone Deep_variant, your file structure should look like this:

```
- gsutil/
- deepvariant/
```

- Run commands:

```
cd deepvariant
./build-prereq.sh
```
